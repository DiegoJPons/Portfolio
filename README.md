# Portfolio

Personal site showcasing full-stack web projects and systems work (Stony Brook CS).

**Live site:** https://portfolio-henna-nine-76.vercel.app/

## Stack

React, TypeScript, Vite, Tailwind CSS

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
```

Output is in `dist/`. Preview the production build:

```bash
npm run preview
```

## Deploy

Connect this repo to [Vercel](https://vercel.com) or [Netlify](https://netlify.com): set the build command to `npm run build` and the publish directory to `dist`.

Project copy and links live in `src/data/projects.ts`.
